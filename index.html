为什么选择 Rabbit？

融合 Python 的简洁、Java 的性能、Rust 的安全

🎯

极简语法

无括号、无分号、无花括号。缩进即结构，代码即意图。新手 30 分钟上手，专家写得更少。

⚡

双模式编译

Python 版快速开发，即时验证想法；C 版极致性能，生产环境部署。一套代码，两种运行。

🛡️

静态类型推断

写 `x = 10`，编译器知道是 Int。错误在编译期捕获，而不是生产环境崩溃。

🚀

高性能并发

轻量级线程 + async/await，轻松处理百万连接。性能对标 Go，语法比 Python 简单。

🔧

零成本抽象

高阶函数、泛型、模式匹配，编译后零开销。用优雅的代码，跑极快的程序。

🌍

民主开源

社区驱动设计，RFC 公开讨论。MIT + Apache 2.0 双许可，个人和企业都能自由使用。
